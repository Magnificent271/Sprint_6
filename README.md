# Тестовый проект 6 спринта

## Структура проекта:
* ### TESTS - папка с тестами
1. #### test_login_user.py - тесты для страницы логина
2. #### test_registration_user.py - тесты для страницы регистрации
3. #### test_create_ad.py - тесты для страницы создания объявления
4. #### test_logout_user.py - тесты для страницы выхода из аккаунта
* ### CONF - папка с конфигурацией
1. #### DATA - файл со вспомогательными функциями данными
* ### LOCATORS - папка с локаторами

### Для запуска использовать команду - pytest
